## Phase 4 — Поддержка других провайдеров LLM (на будущее)

> Цель: не быть привязанным только к Ollama — поддержка облачных и альтернативных провайдеров

| # | Идея | Приоритет |
|---|------|-----------|
| 4.1 | **OpenAI API** (`/v1/chat/completions`) — GPT-4o, GPT-4-mini и др. | высокий |
| 4.2 | **Anthropic Claude API** — claude-3-5-sonnet, claude-haiku и др. | высокий |
| 4.3 | **Google Gemini API** — gemini-1.5-flash, gemini-pro | средний |
| 4.4 | **LM Studio** (`http://localhost:1234/v1/chat/completions`) | средний |
| 4.5 | Универсальный OpenAI-совместимый endpoint (Together AI, Groq, Mistral AI и др.) | средний |
| 4.6 | Хранение API-ключей в `chrome.storage.local` (не в коде) | высокий |
| 4.7 | Выбор провайдера в настройках — переключатель Ollama / OpenAI / Custom | высокий |

> **Примечание:** при добавлении облачных провайдеров данные страницы будут отправляться на внешние серверы. Необходимо явное предупреждение пользователю в UI.
