## Как обновлять

После завершения сессии выполнить:

```bash
# Посмотреть коммиты с временем
git log --pretty=format:"%h %ad %s" --date=format:"%Y-%m-%d %H:%M"

# Посмотреть коммиты за конкретный день
git log --after="2026-03-27 00:00" --before="2026-03-27 23:59" --pretty=format:"%h %ad %s" --date=format:"%H:%M"
```

Затем вручную (или попросив Claude) обновить таблицы выше.
