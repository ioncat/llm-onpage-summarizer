3Последовательность — Пользовательский уровень (простая)

sequenceDiagram participant User participant Extension participant Ollama participant Viewer User->>Extension: Click icon or Right-click activate Extension Extension->>Extension: Extract page text User->>Extension: Pick mode & click Run Extension->>Ollama: Send text + prompt activate Ollama Ollama-->>Extension: Stream response token-by-token deactivate Ollama Extension->>Extension: Display result User->>Extension: Click Expand (⧉) Extension->>Viewer: Open in window/tab activate Viewer User->>Viewer: Ask follow-up questions Viewer->>Ollama: Chat request Ollama-->>Viewer: Response stream Viewer->>Viewer: Display in chat deactivate Viewer deactivate Extension
